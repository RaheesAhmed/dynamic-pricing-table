Installation

  1. Download the Dynamic Pricing Table plugin ZIP file.
  2. Log in to your WordPress admin panel.
  3. Go to Plugins > Add New > Upload Plugin.
  4. Choose the downloaded ZIP file and click "Install Now".
  5. After installation, click "Activate Plugin".

Usage

Once activated, you'll find a new menu item called "Pricing Tables" in your WordPress admin panel. This is where you'll create and manage your pricing tables.

Creating Pricing Tables

  1. Go to Pricing Tables in the admin menu.
  2. Click "Add Pricing Table" or choose a template to start with.
  3. Give your table a title and add pricing tiers.
  4. Customize the colors, styles, and display options.
  5. Click "Save All Tables" to save your changes.

Customization

Each pricing table can be customized with the following options:

  • Background Color
  • Text Color
  • Border Radius
  • Tier Background Color
  • Tier Text Color
  • Button Background Color
  • Button Text Color
  • Display Style (Column or Vertical)

Displaying Pricing Tables

To display a pricing table on your site, use the shortcode provided in the admin interface. For example: 

[dynamic_pricing_table id="0"]

Replace "0" with the ID of the pricing table you want to display.

Templates

The plugin comes with several pre-designed templates to get you started quickly:

  • Default
  • Modern
  • Minimalist
  • Colorful
  • Dark Mode

You can select a template when creating a new table and then customize it to your liking.

FAQ

Can I create multiple pricing tables?

Yes, you can create as many pricing tables as you need.

Can I use different styles for different pricing tables?

Absolutely! Each pricing table can have its own unique style and layout.

Is the plugin responsive?

Yes, the pricing tables are designed to look great on all devices.
